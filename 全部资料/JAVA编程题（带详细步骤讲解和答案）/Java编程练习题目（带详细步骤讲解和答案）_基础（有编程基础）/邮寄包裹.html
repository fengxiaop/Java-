  16版.邮寄包裹

1、实验要求

    本实验要求：根据用户的"包裹参数"计算运费。

    1-1. 运费计算的业务说明：

        1-1.1. 一个包裹的参数包括：重量（kg）、长度（cm）、宽度（cm）、高度（cm）。

        1-1.2. 邮寄包裹的规格需满足：

        重量不大于200kg，长度不大于200cm，宽度不大于200cm，高度不大于200cm。

        1-1.3. 包裹邮费计算规则：

                1）首重是2元。

                2）重量小于100kg，费用 = 续重，0.8元/kg。

                3）重量在100kg~150kg之间，费用 = 续重，0.75元/kg。

                4）重量超过150kg，费用 = 续重，0.7元/kg。

                5）计算规则：运费 = 首重 + 续重。

    1-2. 创建工程并配置环境：

        1-2.1. 限制1. 工程取名：SE_JAVA_EXP_E020。

        1-2.2. 限制2. 创建包，取名：cn.campsg.java.experiment。

        1-2.3. 限制3. 创建类，类名：TestPost。

    1-3. 获取用户输入的包裹参数。

        1-3.1. 提示语句为："输入包裹的数据："

        1-3.2. 重量输入提示语句为："输入包裹的重量(kg)："

        1-3.3. 长度输入提示语句为："输入包裹长度(cm)："

        1-3.4. 宽度输入提示语句为："输入包裹宽度(cm)："

        1-3.5. 高度输入提示语句为："输入包裹高度(cm)："

    1-4. 包裹参数的有效性判断：

        1-4.1. 限制1. 有效性判断函数取名：checkPost。

        1-4.2. 限制2. 有效性判断函数定义于TestPost类内。

        1-4.3. 判断包裹参数：重量、长度、宽度、高度任意数据大于200均返回false。

        1-4.4. 否则返回true。

        1-4.5. 如果包裹参数不符合邮寄规定，则输出："该包裹不符邮寄规定！"

    1-5. 计算运费：

        1-5.2. 限制2. 运费计算函数定义于TestPost类内。

        1-5.3. 根据《包裹邮费计算规则》返回运费计算结果。

    1-6. 打印结果：

        1-6.1. “邮寄重量是：” + 包裹重量 + “（kg）”

public class TestPost {

   public static void main(String[] args) {

 

       // 获取包裹的基本数据（重量、长度、宽度、高度）

       System.out.println("输入包裹的数据：");

       Scanner sc = new Scanner(System.in);

       System.out.print("输入包裹的重量(kg)：");

       float weight = sc.nextFloat();

       System.out.print("输入包裹长度(cm)：");

       float leng = sc.nextFloat();

       System.out.print("输入 包裹宽度(cm)：");

       float width = sc.nextFloat();

       System.out.print("输入 包裹高度(cm)：");

       float height = sc.nextFloat();

       sc.close();

 

       // 判断是否输入包裹合法

       boolean flag = CheckPost(weight, leng, width, height);

       if (!flag) {

           System.out.println("该包裹不符邮寄规定！");

           return;

       }

       double money = getMoney(weight);

       System.out.println("邮寄重量是:" + weight + "(kg)");

       System.out.println("邮寄费用为" + money + "元");

   }

 

   // 判断包袱是否符合规定

   public static boolean CheckPost(float weight, float leng, float width, float height) {

       if (weight > 200 || leng > 200 || width > 200 || height > 200) {

           return false;

       }

       return true;

   }

 

   // 计算费用

   public static double getMoney(double weight) {

       double money = 0;

       if (weight <= 100) {

           money = 2 + weight * 0.8;

       } else if (weight <= 150) {

           money = 2 + weight * 0.75;

       } else {

           money = 2 + weight * 0.7;

       }

       return money;

   }

}
