1、实验要求

    本实验要求：以分组军训为背景，综合体验集合与列表的实际业务应用模式和一般组合。

    1-1. 业务说明：

        1-1.1. 本实验以学校按男女性别分组进行军训为业务背景。

        1-1.2. 常规而言，军训对象包括年级或班级的所有学生集体。 

        1-1.3. 由于男女性别的差异原因，实际军训时需要按性别分组进行。

        1-1.4. 本实验从技术应用的角度来模拟对军训学生集合的分组。

        1-1.5. 分组后，对不同性别组的成员按照身高进行排序。

        1-1.6. 本业务关键概念：学生集合、分组以及排序。

public class Trainee implements Comparable<Trainee> {

  // 姓名

  private String name;

  // 男male和女female

  private String sex;

  // 身高：为按训练者的身高大小进行排序服务

  private float height;

 

  public Trainee() { }

 

  public String getName() {

    return name;

  }

  public void setName(String name) {

    this.name = name;

  }

  public float getHeight() {

    return height;

  }

  public void setHeight(float height) {

    this.height = height;

  }

  public String getSex() {

    return sex;

  }

 

  public void setSex(String sex) {

    this.sex = sex;

  }

  @Override

  public int compareTo(Trainee anthor) {  

    int result = -1;  

  //比较当前对象与传入对象的height属性，如果参数1大于参数2，返回值大于0；如相等,返回值为0；否则返回小于0

    result = Float.compare(this.getHeight(), anthor.getHeight());  

    return result;

  }

}


代码清单：cn.campsg.java.experiment.entity.TraineesGroups.java

public class TraineesGroups {


  /* 军训受训人员集合 */

  private List<Trainee> trnList = null;

  /* 对军训受训人员按性别分组后的结果 */

  private Map<String, List<Trainee>> groups = null;

 

  /**

   * 完成对军训成员的初始化

   *

   * @param trnList

   *            参与军训的成员列表

   */

  public void initTrainees(List<Trainee> trnList) {

    this.trnList = trnList;

  }

 

  /**

   * 如果军训成员数据为空或无成员 ，则直接返回null,不做后续处理

   *

   * @return 分组结果

   */

  public Map<String, List<Trainee>> groupTrainees() {

    if (trnList == null || trnList.size() == 0)

    return null;

    // 创建分组对象

    groups = new HashMap<String, List<Trainee>>();

    groups.put("male", new ArrayList<Trainee>());

    groups.put("female", new ArrayList<Trainee>());

 

    // 对成员按性别分组

    for (int i = 0; i < trnList.size(); i++) {

      // 获取军训人员

      Trainee trainee = trnList.get(i);

      // 分组

      if ("male".equalsIgnoreCase(trainee.getSex()))

        groups.get("male").add(trainee);

      else

        groups.get("female").add(trainee);

    }

    // 对分组受训对象按高矮排序。

    Collections.sort(groups.get("male"));

    Collections.sort(groups.get("female"));

      return groups;

  }

}
