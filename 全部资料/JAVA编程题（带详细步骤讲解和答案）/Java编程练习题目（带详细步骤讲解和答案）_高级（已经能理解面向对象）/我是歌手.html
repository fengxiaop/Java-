代码清单：cn.campsg.java.experiment.entity.AbstractSinger.java

public abstract class AbstractSinger {

   

    public void introduce() {

       System.out.println("Hello, 我是歌手。");

    }

   

    public abstract void sing();

}

代码清单：cn.campsg.java.experiment.entity.PopSinger.java

public class PopSinger extends AbstractSinger{

   

    @Override

    public void sing() {

       System.out.println("我是唱流行乐的。");

    }

   

}

代码清单：cn.campsg.java.experiment.MainClass.java

public class MainClass {

   

    public static void main(String[] args) {

       AbstractSinger s = new PopSinger();

       s.introduce();

       s.sing();

    }  

}
