  16版.邮局派发信件

1、实验要求

    本实验要求：以邮局送件为背景，模拟多人同时派发邮件，体验多线程的实际应用。

    1-1. 业务说明：

        1-1.1. 本实验以实际生活中邮局派发邮件为业务背景。

        1-1.2. 通常邮局每天都会有大量的邮件需要邮递员派发。 

        1-1.3. 根据邮件的性质以及发往地点的不同，安排不同人送件。

        1-1.4. 为了快速处理好每日的发件工作，邮局需要安排多人同时发件。

        1-1.5. 每个发件人每日定量发件，并将邮件逐一派发给不同收件人。

        1-1.6. 本业务关键概念：邮递员、派件数量以及同时发件。

    1-2. 创建工程并配置环境：

        1-2.1. 限制1. 创建实验工程，命名为：SE_JAVA_EXP_E045；

        2-2.1. 在包cn.campsg.java.experiment.entity 中创建邮递员类：Postman；

        2-2.2. 为Postman类定义属性：

    2-3. 创建邮局送信业务类:

        2-3.1. 在包cn.campsg.java.experiment.entity中创建送信业务类SendMails：

                1） 该类需继承线程类Thread，并且重写Thread的run方法。

        2-3.2. 为SendMails类创建代表邮递员的私有属性： 

                1） 定义属性：Postman post;

                2） 为属性实现对应的setter/getter方法；

  +提示

 }

        2-3.4. 为SendMails类重写继承而来的run方法，实现发件功能：

                1） 定义发信计数器变量：int count=0；

                2） 如果当前邮递员送件数量大于0，则持续循环送件：

                    其一，在循环体内向控制台输出如下信息：

                        邮递员姓名 +" 还有"+送件数量+ "封信。开始送第 " + (++count) + "封信！"

                    其二，输出信息后，暂停1秒：Thread.sleep(1000)；

                    其三，更新待送信件数量：

                        post.setMailCount(post.getMailCount()-1);

                    其四，判断待送信件数量，若小于等于0，则输出如下信息：

                        邮递员姓名+“ 已完成所有邮件派送！。”

2.代码清单：cn.campsg.java.experiment.entity.SendMails.java

public class SendMails extends Thread {

    // 邮局送信人员：

    private Postman post;

    public Postman getPost() {

       return post;

    }

    public void setPost(Postman post) {

       this.post = post;

    }

    public SendMails() {

    }

    public void run() {

       int mCount = 0;

       while (post.getMailCount() > 0) {

           System.out.println(post.getPostName() +" 还有"+post.getMailCount()+ "封信。开始送第 " + (++mCount) + "封信！");          

           try {// 暂停1秒

              Thread.sleep(1000);

           } catch (InterruptedException ie) {

              System.out.println("线程中断："+ie.getMessage());

           }

           post.setMailCount(post.getMailCount()-1);

           if(post.getMailCount()<=0){

              System.out.println(post.getPostName()+" 已完成所有邮件派送！");

           }

       }

    }

}
