1、数组的概念和C、C++一致，一串连续的地址空间

2、数组的定义
1) int[] a=new int[5];//或者int a[]=new int[5];
2) int a[];	a=new int[5];
3) int[] a={1,2,3};
4) int[] a=new int[]{1,2,3};

3、数组重要属性 length：获得数组元素个数

4、数组赋值：int[] b=a;	//含义是a、b两个数组引用指向“同一个数组首地址”
即此时，a数组和b数组是同一个数组。
故执行b[0]=100;语句后，a[0]的值也将是100.

5、二维数据
int[][] c=new int[3][4];
其中
1)c.length的结果是3，代表行数（一维元素个数）
2)c[1].length的结果是4，代表列数(第2行元素个数 二维元素个数)

6、不规则二维数据组
int[][] d=new int[3][];
d[0]=new int[1];
d[1]=new int[3];
d[2]=new int[5];
